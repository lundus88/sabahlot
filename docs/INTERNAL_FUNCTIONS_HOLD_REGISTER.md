## Release Gate

These functions can only be considered for limited release after:

1. Minimum 10 valid user feedback responses received.
2. No critical bug reported by Alpha testers.
3. Draw / Save / Export basic workflow is accepted by users.
4. Users understand that SabahLot output is preliminary only.
5. Privacy and data handling remain acceptable.
6. Project Director approval is given before exposing any internal module.

## Important Disclaimer

All advanced land planning functions remain preliminary. They are not official survey plans, not legal boundary evidence, not ownership confirmation, and not approval from JTU, Pejabat Tanah, PBT, or any authority.
