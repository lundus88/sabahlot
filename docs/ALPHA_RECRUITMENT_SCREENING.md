# SabahLot Alpha Recruitment Screening

Purpose: select suitable additional users for the Controlled Alpha group.

## Recruitment Goal

Target:
- Complete 10 Controlled Alpha users.
- Recruit 5 additional users if existing users are slow to respond.
- Keep access controlled and private.

## Suitable User Criteria

| Criteria | Preferred |
|---|---|
| Landowner / family land user | Yes |
| Orang kampung / komuniti setempat | Yes |
| Has smartphone | Yes |
| Can use map apps | Preferred |
| Willing to give feedback | Required |
| Understands preliminary output | Required |
| Will not treat output as official boundary | Required |
| Will not share link publicly | Required |

## User Type Balance

Try to include:

| User Type | Target |
|---|---:|
| Pemilik tanah / ahli keluarga | 3–5 |
| Orang kampung / komuniti setempat | 2–3 |
| Pengguna teknikal / mapping | 1–2 |
| Listing partner candidate | 1–2 |

## Screening Questions

Before sending the SabahLot Alpha link, ask:

1. Adakah tuan/puan pemilik tanah atau ahli keluarga pemilik tanah?
2. Di daerah / kawasan mana tanah tersebut berada?
3. Adakah tuan/puan boleh mencuba menggunakan telefon atau laptop?
4. Adakah tuan/puan sanggup beri feedback selepas mencuba?
5. Adakah tuan/puan faham bahawa SabahLot bukan pelan ukur sah?
6. Adakah tuan/puan faham bahawa SabahLot bukan bukti sempadan rasmi?
7. Adakah tuan/puan setuju untuk tidak sebarkan link kepada umum dahulu?

## Risk Control Checklist

Do not send the Alpha link if:

- User expects official boundary confirmation.
- User asks to confirm ownership or title status.
- User wants SabahLot to replace a licensed surveyor.
- User wants to use output for legal dispute.
- User refuses to accept preliminary disclaimer.
- User wants to share the link publicly.

## Safe Acceptance Message

Terima kasih. Berdasarkan maklumat tuan/puan, saya akan masukkan tuan/puan sebagai pengguna SabahLot Alpha.

Mohon guna untuk ujian awal sahaja. Semua output adalah preliminary dan bukan dokumen rasmi.

Saya akan hantar link dan borang feedback secara private.

## Decline / Hold Message

Terima kasih kerana berminat. Untuk masa ini SabahLot masih dalam ujian Alpha yang sangat terhad.

Saya akan simpan nama tuan/puan untuk fasa ujian seterusnya apabila sistem lebih stabil.

## Manual Review Triggers

Flag for manual review if the user mentions:

- NCR
- tuntutan adat
- sempadan bertindih
- pertikaian sempadan
- waris
- pecah geran
- status hakmilik tidak jelas
- geran hilang
- pembeli / penjual tanah
- listing tanah pihak ketiga

## Non-Negotiable Disclaimer

SabahLot masih dalam fasa ujian Alpha. Semua output adalah preliminary sahaja. Ia bukan pelan ukur sah, bukan bukti sempadan rasmi, bukan pengesahan hakmilik tanah, bukan kelulusan pecah geran, dan bukan kelulusan mana-mana pihak berkuasa.

Semua sempadan, keluasan dan status tanah perlu disahkan oleh pihak berkuasa berkaitan, juruukur berlesen atau penasihat profesional sebelum digunakan untuk tujuan rasmi.
